# Security Assessment Platform - Frontend

## Giới thiệu
Frontend cho nền tảng đánh giá an ninh mạng, được xây dựng bằng Vue.js 3.

## Công nghệ sử dụng
- Vue.js 3
- Vite
- Pinia (State Management)
- Vue Router
- Axios
- Socket.IO Client
- Chart.js
- TailwindCSS / Bootstrap 5

## Cài đặt

### Yêu cầu
- Node.js >= 18
- NPM >= 9

### Các bước cài đặt
```bash
# Clone repository
git clone <repository-url>
cd frontend

# Cài dependencies
npm install

# Chạy môi trường development
npm run dev

# Build production
npm run build

# Preview build
npm run preview
